# Basic Github
## Work Flow
### 1. Pull previous edits down
```
git pull
```
### 2. Edit your file, then add all the edits
```
git add *
```
### 3. Commit your edits
```
git commit -m "YOUR MESSAGE HERE"
```
### 4. Push your edits
```
git push
```
### 5. If push fails due to merge error, follow the directions after merging
```
git merge
```
